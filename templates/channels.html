Add channels

Paste a channel link, or import a .txt / .csv with one or more links per line.

Accepts youtube.com/@handle, /channel/UC… links, or a bare @handle.

Every YouTube link or @handle found in the file gets added. Optional columns after the link set niche, sub-niche and agency (e.g. youtube.com/@x, Tech, Mini PCs, BossMgmtGrp); with a header row like channel,agency you can supply just the columns you have. Re-importing existing channels fills in blank fields without creating duplicates — values you've set by hand are never overwritten.

Backfill scan

The regular Scan now only covers each channel's newest uploads. A backfill walks the full upload history of every tracked channel back to the chosen cutoff, skipping videos already in the database.

Fetching a description takes a second or two per video, plus a built-in polite delay to avoid YouTube rate limiting — a deep backfill over many channels can run for hours. It's safe to do this once and let regular scans take over from there.
